Para compilar:
g++ -std=c++11 main.cpp -o main

Para ejecutar:
El programa extrae la información automáticamente de los archivos provistos.

Si se quiere cambiar el archivo de entrenamiento, debe nombrarse training_set.txt 
Cada mensaje debe estar en una linea y la linea debe iniciar con h o s, dependiendo si es spam o ham

Si se quiere cambiar el archivo de prueba, debe nombrarse test.txt 
Cada mensaje debe estar en una linea y la linea debe iniciar con h o s, dependiendo si es spam o ham
